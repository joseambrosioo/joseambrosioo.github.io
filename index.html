Criptografia AES e RSA

Python | Streamlit Cloud | Visual Studio Code

Este aplicativo baseado em Python que criptografa e descriptografa mensagens, suportando as cifras AES e RSA.

Desenvolvi os algoritmos de criptografia e descriptografia do zero, criando um sistema que mostra cada etapa do processo AES, incluindo a chave inicial, chaves de rodada e texto cifrado para cada bloco de 16 bytes.

A cifra AES usa um tamanho de chave de 256 bits, e a cifra RSA pode gerar pares de chaves públicas (n, e) e pares de chaves privadas (n, d) com 80 ou 128 bits.

Previsão de Calorias Queimadas

Python | Análise de Dados | Machine Learning | TensorFlow | Numpy | Pandas | Matplotlib | Seaborn | Scikit-Learn | Plotly | Jupyter Notebooks | Kaggle | conjunto de dados fmendes-DAT263x-demos

Desenvolvi um modelo de machine learning para prever as calorias queimadas durante actividades físicas, alcançando um desempenho impressionante com métricas que incluem 1.003 de Erro Médio Absoluto (MAE), 0.999 de Coeficiente de Determinação (R²), 2.035 de Erro Quadrático Médio (MSE) e 1.426 de Raiz do Erro Quadrático Médio (RMSE), superando os padrões de referência.

Usei um conjunto de dados de 15.000 registros, utilizando a duração da actividade, a frequência cardíaca e a temperatura corporal como preditores. Conduzi uma análise exploratória de dados (EDA) para descobrir padrões, tendências e anomalias através de visualizações, estatísticas descritivas e análise de correlação.

Implementei múltiplos modelos de regressão, incluindo XGBoost, Extra Trees, Perceptron Multicamadas (MLP), Random Forest, Gradient Boosting, K-Nearest Neighbors (KNN), Linear, Lasso, Ridge, SVM, ElasticNet, Decision Tree, Huber e Bayesian Ridge, para identificar a abordagem ideal.
